## Comandos Screen básicos

### Obter versão
	screen -vesion

### Ativar screen com sessão sem nome
	screen 

### Criar sessão com nome session1
 	screen -S session1

### Listar sessões
	screen -ls

### Retomar sessão
	screen -r [número_da_sessão]  //Retomar sessão

### Salvar a sessão e sair deixando-a em execução
	Ctrl + a + d

### Sair e fechar a sessão
	Exit ou Ctrl + d
