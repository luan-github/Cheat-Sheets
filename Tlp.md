### Arquivo de Configuração:
	/etc/default/tlp

### Aplicar configuração
	sudo tlp start

### Alternar para o modo de bateria (independentemente da configuração)
	sudo tlp bat

### Alternar para o modo de carga (independentemente da configuração)
  sudo tlp ac

### Status
    $ sudo tlp-stat [-s|-c|-t|-b]
      -s - System Info
      -c - Configuration
      -t - Temperature
      -b - Battery
